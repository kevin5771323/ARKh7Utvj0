# 项目简介

## 核心功能
- 提供基于YOLOv8的墙面脱落检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 支持选择是否显示标签和原图

## 适用场景
- 建筑物墙面安全检测
- 墙面脱落问题自动化识别
- 大规模墙面状况筛查
- 实时墙面监测

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Python3.9开发环境
- 集成OpenCV-Python和PyQt5
- 包含3000张标注数据集，涵盖3类墙面问题
- 提供完整源码、训练好的模型及运行教程
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
